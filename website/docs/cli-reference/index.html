CLI Reference

Updated 2026-03-24

Complete command reference for the Hookwing CLI. Manage endpoints, events, deliveries, and API keys from the terminal.

Installation

npm install -g @hookwing/cli

Authentication

Set your API key as an environment variable:

export HOOKWING_API_KEY=hk_live_your_key

Or pass it directly:

hookwing --api-key hk_live_your_key endpoints list

Commands

hookwing endpoints list

List all endpoints in your workspace.

hookwing endpoints list

Options:

  • --limit &lt;n&gt; — Maximum results (default: 20)
  • --json — Output as JSON

hookwing endpoints create

Create a new webhook endpoint.

hookwing endpoints create \
  --url https://your-app.com/webhooks \
  --name my-endpoint \
  --event-types payment_intent.succeeded,charge.failed

Options:

  • --url &lt;url&gt; — Destination URL (required)
  • --name &lt;name&gt; — Human-readable name (required)
  • --event-types &lt;types&gt; — Comma-separated event type filters

hookwing endpoints get &lt;id&gt;

Get details for a specific endpoint.

hookwing endpoints get ep_abc123

hookwing endpoints delete &lt;id&gt;

Delete an endpoint.

hookwing endpoints delete ep_abc123

hookwing events list

List recent events.

hookwing events list --limit 10

Options:

  • --limit &lt;n&gt; — Maximum results (default: 20)
  • --since &lt;timestamp&gt; — Only events after this timestamp
  • --event-type &lt;type&gt; — Filter by event type

hookwing events get &lt;id&gt;

Get event details including payload.

hookwing events get evt_xyz789

hookwing events replay &lt;id&gt;

Replay an event (re-deliver to all matching endpoints).

hookwing events replay evt_xyz789

hookwing deliveries list

List delivery attempts.

hookwing deliveries list --limit 20

Options:

  • --limit &lt;n&gt; — Maximum results
  • --status &lt;status&gt; — Filter: delivered, failed, pending

hookwing keys list

List API keys for your workspace.

hookwing keys list

hookwing keys create

Create a new API key.

hookwing keys create \
  --name "CI/CD key" \
  --scopes endpoints:read,events:read

hookwing keys delete &lt;id&gt;

Revoke an API key.

hookwing keys delete key_abc123

Global Options

FlagDescription
--api-key &lt;key&gt;API key (overrides HOOKWING_API_KEY)
--base-url &lt;url&gt;API base URL (default: https://api.hookwing.com)
--jsonOutput as JSON
